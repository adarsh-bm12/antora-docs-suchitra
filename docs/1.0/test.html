Field Description

  • redirecturl string

    Call back URL

  • UniqueId string

    Invoice Unique Id

  • TransactionType string

    Transaction type: Invoice or Return

  • TransactionCode string

    transaction Code

  • LocationCode string

    Location code defined through this page will be auto fetched else we have to manually provide in code.

  • ReferenceId string

    Unique Id For Reference

  • CurrencyCode string

    Invoice currency Code

  • Note string

    Invoice Note

  • Number string

    Invoice Number

  • Date string

    Invoice date format should be dd-mm-yyyy (28-12-2023)

  • ItemTotal decimal

    Product total amount

  • Discount decimal

    Header discount

  • DiscountPercentage decimal

    Header discount percentage (If we provide discount percentage, also provide calculated value in discount)

  • TaxesIncluded string

    If Tax Included set "Y" for Yes and "N" for No

  • SubTotal decimal

    Sub total Amount

  • Tax: Invoice tax details

    • UniqueId string

      TAX Unique Id

    • TaxCode string

      TAX Code

    • Percentage decimal

      TAX Percentage

    • Value decimal

      TAX Value

  • TotalTax decimal

    Total tax amount for the invoice

  • TotalAmount decimal

    Total Amount of the invoice

  • Items: Item details

    • UniqueId string

      Product Unique Id

    • ProductTitle string

      Product Title

    • SKU string

      Product barcode

    • Quantity decimal

      Product Quantity

    • Price decimal

      Product Unit Price

    • SubTotal decimal

      Product sub total (Quantity*Price)

    • TotalTax decimal

      Product Total Tax Amount

    • TaxesIncluded string

      Tax is included for the product

    • Tax: Product Tax Details

      • UniqueId string

        TAX Unique Id

      • TaxCode string

        TAX Code

      • Percentage decimal

        TAX Percentage

      • Value decimal

        TAX Value

    • Discount decimal

      Discount Amount for the item

    • DiscountPercentage decimal

      Discount Percentage for the item

    • TotalAmount decimal

      Product Total Amount

  • Customer: Customer inputs

    • UniqueId string

      Customer Unique Id

    • Name string

      Customer Name

    • Phone string

      Customer Phone Number

    • Email string

      Customer Email Id

    • Address: Customer address inputs

      • Address1 string

        Address Line 1

      • Address2 string

        Address Line 2

      • City string

        City name of the Customer

      • Country string

        Country name of the Customer

      • Area string

        Area Name of the Customer

    • UseSameDeliveryAddress string

      set "Y" for same delivery address else set "N"

    • DeliveryAddress: Delivery address details

      • Address1 string

        Address Line 1

      • Address2 string

        Address Line 2

      • City string

        City name

      • Country string

        Country Name

      • Area string

        Area Name

  • Salesman: Salesman Details

    • UniqueId string

      Salesman Unique Id

    • Name string

      Salesman Name

  • Payment: Payment Details

    • UniqueId string

      Payment Unique Id

    • Type string

      Payment Type Name

    • Amount decimal

      Pay Amount

    • Tender decimal

      Tender Amount

    • Change decimal

      Change Amount

    • Reference string

      Payment Reference
